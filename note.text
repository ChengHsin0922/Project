重置資料表id:
            在資料表依序執行以下SQL語法:
            SET  @num := 0;
            UPDATE 資料表名稱 SET id欄位 = @num := (@num+1);
            
            ALTER TABLE 資料表名稱 AUTO_INCREMENT =1;
